# Stellar Suite Promotional Assets

This directory contains the promotional banners and design assets for the Stellar Suite VS Code Marketplace listing.

## Contents
- `GUIDELINES.md`: Design specifications, color palettes, and typography rules.
- `source/`: Source files and generation tools.
- `variations/`: Pre-rendered SVG variations of the banner for different platforms.

## Generating PNG Assets
Since SVG rendering across different markdown parsers can vary, you can export these SVGs to PNG format using the provided utility:

1. Open `source/banner-generator.html` in your web browser.
2. The page will display the SVGs and provide "Download PNG" buttons for each variation.

## Variants Included
1. **Marketplace Banner (1024x256)**: Designed to sit at the top of the extension's README.
2. **Social Card (1280x640)**: Designed for GitHub repository OpenGraph image, Twitter, and other social sharing.
3. **Sidebar Banner (300x250)**: Compact format for documentation sidebars or smaller feature boxes.
